# dotfiles
Dotfile for arch+i3+polybar setup

## Packages
1. i3-gaps
2. lightdm
3. lightdm-gtk-greeter
4. betterlockscreen
5. playerctl
6. polybar
7. pywal(optional)
8. mpd(optional)
9. ncmpcpp(optional)
10. urxvt(rxvt-unicode)
11. compton
12. feh
13. lxapperance(optiona: to choose themes)
14. arc theme
15. papirus icon pack
16. rofi
17. ranger(cmd based), thunar(gui)
18. neofetch(optional)
